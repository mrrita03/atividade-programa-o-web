# 📚 Exercícios de Programação Web

Este repositório contém os exercícios práticos desenvolvidos durante as aulas de **Programação WEB** no **Senac SP**.

## 🚀 O que foi desenvolvido:
Neste projeto, apliquei conceitos fundamentais de HTML5 e CSS3, incluindo:
- **Estruturação de Textos:** Uso correto de tags de cabeçalho e parágrafos.
- **Listas e Receitas:** Organização de informações ordenadas e não ordenadas.
- **Tabelas:** Construção de estruturas de dados tabulares.
- **Formulários:** Criação de campos de entrada e identificação de usuário.
- **Geografia e Listas:** Exercício focado em América do Sul.

## 🛠 Tecnologias Utilizadas:
- **HTML5**: Estruturação semântica.
- **CSS3**: Estilização e layout.
- **VS Code**: Editor de código utilizado.
